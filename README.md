# Chatbot Senac - Calculadora de IMC com Gemini

Um chatbot descontraído e bem-humorado que calcula o **Índice de Massa Corporal (IMC)** usando o Google Gemini (gemini-2.5-flash).

Desenvolvido como projeto prático para demonstrar integração com a API do Google GenAI.

## 🎯 Sobre o Projeto

O "Chatbot Senaczinho" é um assistente virtual com **personalidade de rua**, bem humorado e direto ao ponto. Ele pede seu peso e altura, calcula seu IMC e te dá uma análise sincera e engraçada sobre o resultado.

**Características principais:**
- Integração com Google Gemini 2.5 Flash
- System prompt personalizado para respostas descontraídas
- Interface simples via terminal
- Histórico de conversa mantido

## 🚀 Como Executar

### Pré-requisitos
- Python 3.8 ou superior
- Chave de API do Google Gemini

### Instalação

Você: Oi
Chatbot senaczinho: E aí, beleza? Bora calcular esse IMC? Me fala teu peso (em kg) e tua altura (em metros).

Você: 85 1.75
Chatbot senaczinho: Caralho, tu tá com 27.8... Tá no sobrepeso leve, irmão. Tá na hora de largar o refri e dar uma caminhada, hein? kkk
